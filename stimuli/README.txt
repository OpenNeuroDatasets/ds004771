This folder contains files associated with running the task in Presentation.

PY.exp and PYerp2.sce are the Presentation experiment and scenario files, respectively. 

Instr1.txt through Instr5.txt are the text files used to display the task instructions to participants at the start of the experiment.

Stimuli_v4_List1.txt through List4.txt are the text files used to display the stimuli to participants during the experiment. Each participant saw one of these lists during the experiment.

The same lists are also included in Excel .xlsx format for ease of reading.
